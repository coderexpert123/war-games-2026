# Iran War 2026 Analysis — Project Instructions

## What This Project Is
A comprehensive geopolitical and resource-level simulation of the 2026 Iran War (US/Israel vs Iran, started Feb 28 2026). Built from parallel research agents, resource-by-resource deep dives, and combinatorial cascade modeling.

## How to Continue This Work

### Research Methodology
- Deploy parallel sub-agents per resource or country for maximum depth
- All claims must be sourced with publication name and date
- Distinguish verified facts from reasonable inferences from speculation
- Update simulation as new data becomes available

### Key Files
- `simulation/master-simulation.md` — The integrated prediction. Update this when new data changes probabilities.
- `cascades/combinatorial-matrix.md` — The resource-industry-country dependency web. This is the analytical core.
- `blind-spots/analysis.md` — Known gaps. Work through these systematically.

### Priority Research Gaps (from blind-spots/analysis.md)
1. Pakistan, North Korea, Japan, South Korea — underweighted actors
2. Cyber warfare escalation scenarios
3. Water crisis modeling (desalination attacks)
4. Pharmaceutical supply chains
5. November 2026 convergence (midterms + Chinese leverage + winter)
6. Insurance systemic risk

### Conventions
- Dates in ISO format or Month Day, Year
- All prices in USD unless noted
- Production in barrels per day (bpd) for oil, metric tonnes for minerals
- Sources at bottom of each file

## Intelligence Layers (Agentic Brain)
1. **Raw data** — Resource production figures, prices, trade flows → `/resources/`
2. **Country analysis** — Incentives, positions, actions → `/countries/`
3. **Industry impact** — Semiconductors, AI, defense, food → `/industries/`
4. **Cascade modeling** — How disruptions compound → `/cascades/`
5. **Integrated simulation** — Predictions and scenarios → `/simulation/`
6. **Gaps and uncertainties** — What we don't know → `/blind-spots/`
