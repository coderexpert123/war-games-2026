# 2026 Iran War: Comprehensive Geopolitical & Resource Analysis

**Date of analysis:** March 23, 2026
**Status:** War in fourth week. US/Israel vs Iran. Ceasefire talks initiated.

## What This Is

A multi-dimensional geopolitical simulation built from granular resource-level data, country-by-country incentive analysis, and combinatorial cascade modeling. All claims are sourced from major news outlets, think tanks, and government reports dated February-March 2026.

## Repository Structure

```
/countries/          — Country-by-country incentive analysis
/resources/          — Resource-by-resource supply chain deep dives
/industries/         — Industry impact analysis (semiconductors, AI, defense, food)
/cascades/           — Combinatorial cascade models (resource → industry → country)
/simulation/         — Integrated simulation and predictions
/blind-spots/        — Known gaps, uncertainties, and unmodeled risks
```

## Methodology

- 8 parallel research agents deployed per resource
- 8 parallel research agents deployed per country/region
- Combinatorial analysis identifying correlated disruption cascades
- Simulation built bottom-up from resource constraints, not top-down from political assumptions

## Key Findings

1. **China's resource position is the dominant strategic fact** — controls rare earths, processing, and is insulated from Hormuz
2. **The chip famine is real and structural** — helium (33% offline), bromine (67% at risk), LNG (Taiwan 11-day reserve)
3. **Food crisis is locked in** — Northern Hemisphere spring planting already affected; yields will disappoint Q3 2026
4. **Insurance outlasts the war** — Lloyd's war-risk designation persists weeks/months after ceasefire
5. **US munitions depletion weakens China deterrence** — the real strategic cost of this war

## Caveat

All analysis based on open-source reporting. Wars are inherently unpredictable. A single escalation could redirect everything.
